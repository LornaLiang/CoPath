# CoPath Demo 数据设计说明

## 1. 文档目的

本文档用于定义 CoPath Demo 系统所需的全部演示数据。

目标：

- 保证所有页面都有真实数据展示
- 保证整个 Demo 演示流程完整
- 为数据库初始化提供依据
- 为 seed_data.py 提供数据来源

本文件不是数据库设计文档，而是 Demo 场景设计文档。

# 14. 设计原则

1. 所有 Demo 数据必须来源于本文件。
2. 所有数据库初始化数据必须与本文件一致。
3. 所有页面展示数据必须能够在本文件找到对应来源。
4. 所有 AI 对话必须服务于动态路径调整演示。
5. Demo 数据应体现不同学生、不同路径、不同调整策略。
